# Tata iQ — AI-Powered Collections Strategy (Geldium)

## Overview
Virtual internship simulation completed as part of the Tata iQ 
Data & AI Strategy program. I worked as an AI transformation 
consultant advising Geldium, a digital lending company, on 
building an AI-powered credit delinquency prediction and 
collections intervention system.

## My Role
AI Transformation Consultant — Tata iQ Financial Services 
Data & AI Strategy Team

## Project Tasks

### Task 1 — Exploratory Data Analysis (EDA)
- Analyzed a 500-customer dataset (19 variables)
- Identified missing values, data quality issues, and anomalies
- Key finding: flagged near-zero correlation between standard 
  risk variables and the delinquency target — a critical 
  data integrity issue surfaced before modeling began
- Tools: Python (pandas), GenAI summarization

### Task 2 — Predictive Modeling Plan
- Designed a logistic regression-based delinquency 
  prediction framework
- Justified model choice on interpretability and regulatory 
  compliance grounds
- Defined evaluation strategy: F1 score, AUC-ROC, 
  disparate impact analysis

### Task 3 — Business Report & Data Storytelling
- Translated technical findings into a stakeholder report 
  for Geldium's Head of Collections
- Developed a SMART intervention recommendation: 
  6-week SMS + agent outreach pilot targeting high-risk 
  customers before the 30-day delinquency threshold
- Included responsible AI ethics section covering fairness, 
  explainability, and regulatory compliance

### Task 4 — Executive Presentation
- Built a 6-slide PowerPoint deck for Geldium's leadership
- Covered system workflow, agentic AI roles, 
  responsible AI guardrails, and expected business impact
